# 🎯 Spectrack

מערכת ניהול מידע למכשירים בתחום הספקטרום האלקטרומגנטי  
(למשל: רחפנים, מכשירי קשר, לוויינים, רדיו ומערכות הפעלה מרחוק).

המערכת מאפשרת:

- תיעוד מלא של מכשירים
- קטלוג לפי תדרים
- תחקירים
- ניתוחים טכניים

---

## 🚀 טכנולוגיות

- ⚛️ React + TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧠 Git + GitHub

---

## ⚙️ התקנה והרצה מקומית

```bash
npm install
npm run dev
```
